This site is a HTML version of HelpPC Reference Library, a DOS-based hypertext program by David Jurgens. The conversion program was written by Stanislav Sokolov and the C source is available for download under GNU General Public Licence. In the process of conversion several dead links and inconsistencies were fixed in the original library. You can download the off-line version of the library, compressed in RAR format (221K). The archive includes the original HelpPC for DOS and the program (with source) to convert it to HTML. The original distribution of the library can be found here.
